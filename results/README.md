# Results

This directory contains key result metrics. Large outputs (tif raster maps, full response CSVs, etc.) are excluded from the repository due to size constraints. They can be regenerated by running the scripts.

## Included

| File | Content |
|:-----|:--------|
| `significance_tests.json` | DeLong test Z/p-values between all model pairs |
| `maxent_metrics.json` | MaxEnt (R maxnet) per-fold AUC + means |

## Reproducing Full Results

### Response curves (PDP)
```bash
cd scripts
python3 08_train_kan_full_interpret.py   # Train full-data KAN
python3 10_export_response_data.py       # Export PDP CSVs to Results/kan_response_data/
```

### Maps (suitability rasters)
```bash
python3 16_generate_current_suitability_maps.py
python3 18_generate_other_baseline_maps.py
```

### Future projections
```bash
python3 21_future_projections.py
```

### Figures
All figure scripts read from intermediate CSVs/rasters and write to `manuscript/figures/`.
See `scripts/README.md` for the full pipeline order.
